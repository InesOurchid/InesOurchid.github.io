La notion de fichier

    Le fichier - Feuille


Ce fichier est composé d’une base (un nom) + d’un . + une extension Une base est une suite de lettres, chiffres et quelques symboles dont ( - ; _ ) ex : Controle_MDIND_2022 L’extension : porte l’information ex : .png, .jpeg, .docx, .pdf etc. Dans ce cours nous retrouverons les extensions .txt, .md (morkdown —> Format d’écriture qui permet dans le fichier de mettre en forme), .py (python), .html et .css
